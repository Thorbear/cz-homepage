Welcome to ChatZilla's developer area; here we plan to include everything necessary to develop for ChatZilla - whether that means writing scripts, writing motifs or finding and fixing bugs.

  • API (Application Programming Interface): Documentation for the various JavaScript Prototypes in ChatZilla and all their methods and arguments. This is generated directly from the code, so remarks are currently a bit patchy, but will improve over time.
  • #chatzilla: Our IRC channel is always open for script, motif and other questions. Feel free to drop-in if something doesn't make sense, you can't the right API or you just want to chat about ChatZilla's internals.
