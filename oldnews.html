Older News

September 5, 2004
  • ChatZilla 0.9.64g fixes the memory problems some people have experienced with 0.9.64f, as well as many other fixes and improvements.
August 25, 2004
  • ChatZilla 0.9.64f has been causing problems for some people. If you are experiencing excessive memory or CPU usage when running 0.9.64f you are advised to install 0.9.64e until we work out the exact cause and fix it properly. If you need the new features in 0.9.64f (SSL and full Unicode nicknames), please check back regularly for new revisions that solve this problem.
June 10, 2004
  • Firefox 0.9 support added. With Firefox 0.9 being released in just a few days, we've now fixed the majority of issues with ChatZilla running on current versions of the Mozilla Firefox browser. Update, June 17, 2004: some more compatibility fixes have been made, including fixing opening URLs in a new window. Please get the latest revision to try it out.
  • Mozilla blocks XPInstall from "untrusted" hosts. Bug 240552 blocks XPInstall from "untrusted" hosts, which means you can no longer install ChatZilla directly from this website with Mozilla 1.7 and newer. Instead, you need to download the XPI files and open them in Mozilla on your computer.
April 22, 2004
  • ChatZilla now supports DCC File transfers. Yes, it's finally here, but don't expect it to be perfect just yet - it's still very much a work-in-progress feature. Download the latest revision if you want to try it out.
December 25, 2003
  • ChatZilla's preference window now works in Mozilla Firefox. It took some time, but it's finally working. Anyone who uses ChatZilla in Mozilla Firefox should check out version 0.9.54c or later, and we'd love to hear suggestions for making it better. Note: all versions since 0.9.63 have this feature as standard, and work in the Mozilla Suite and Mozilla Firefox.
November 17, 2003
  • ChatZilla has a new web page. "You're soaking in it."™ Check out the new motifs page when you have a minute. Returning visitors will notice that only recent revisions are shown in the table below. You can still access the complete list on the Revision History page. I've also added a way to donate via PayPal, in case you want to contribute to the cause.
  • James Ross gets CVS access. James is the creator the ChatZilla pref panels, RPL_ISUPPORT support, and halfop mode support. He also contributes regurlarly in Bugzilla and on #chatzilla, as Silver.
  • Server sockets are in! Darin Fisher has added server sockets to Necko in the latest nightlies. This makes it possible to add DCC support to ChatZilla. See bugs 92928 and 73257 for more information.
